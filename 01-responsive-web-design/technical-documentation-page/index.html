Introduction

This documentation is about an invented language, dessL, which is a simplified interpreted language, inspired by python and very close to pseudocode.

As of this writing, the language is not created yet, therefore you will not be able to test it, only digest it.

To start off, let's observe how the comments in dessL are simple:

start print "Hello World" -- this is a comment and will not be processed - multi-line comments are as simple as adding only 1 dash -- and this would end the multi-line comments. end
Variables

desseL proposes a very intuitive way to store data into variables. Since it is taking its roots in python, you can assign a variable without declaring its type. This simplifies the work for the programmer as everything is done behind the scenes. For example, if you want to assign 10 to the variable number, you would proceeed as such:

start 10 in number end

As you can see, the program always starts with the start keyword and ends with the end keyword. For strings, the assignation goes like this:

start "Dependency over double quotes" in myString end

It is worthy of note to state that the variables are case-sensitive, as in most languages. Hence, myString is different than my_string or mystring. Per convention, we cannot start a variable with a number. Therefore, 5myString would not be valid.

Note that there is a distinction between single quotes and double quotes in dessL. Since, as you will see below, the single quotes are used in looping, you cannot use single quotes for strings.

Conditions

Conditions are very simple in dessL. For example, the following will print Hello World

start true in myBool if myBool print "Hello World" end

Evidently, adding an else is simple too:

start false in myBool if myBool print "Hello World" else print "Goodbye World" end
Loops

Loops in dessL have taken the most simple form possible. One only has to use the reserved loop keyword and declare the loop as in the following example that prints to the console "Hello World" 5 times:

start 'loop 5 print' "Hello World" end

Another example, this time adding 1 to a variable 5 times:

start 0 in myVariable 'loop 5 add' myVariable 1 end

As you will see in the reserved words, all arithmetic operations available in other languages are available in dessL.

-- the following code will return to the console 0 five times. start 8 in myVariable 'loop 5 mod' myVariable 2 end
Arrays

Arrays in dessL are quite like most languages:

start [1,2,3,4,5] in myArray print myArray end -- this would print 1,2,3,4,5

Using loops in arrays:

start [1,2,3,4,5] in myArray 'loop myArray.length print' myArray end -- this would print - 1 - 2 - 3 - 4 - 5 -- looping over the entire array.
Functions

Functions in dessL are created with the word func and are called like most languages with the ().

start func = print "Hello World" func() end -- this prints Hello World

You can pass in argument to the function also:

start func(x, y) = x add y return func(2, 3) end -- this returns 5.

Notice in the last example the return at the end to return the value. It this example, since there was only one operation inside the function, dessL evaluates the return automatically. However, in the following example, a value mus be added for the return to be evaluated:

start func(x, y) = 2 in toReturn, x add y, return toReturn func(2, 3) end -- this returns 2.
Reserved Words

You can find the list of reserved words in dessL below. Please note that since this language is still under development, new reserved words can be added from time to time.

  • start
  • print
  • end
  • in
  • true
  • false
  • if
  • loop
  • else
  • add
  • sub
  • mult
  • div
  • div
  • mod
  • func
